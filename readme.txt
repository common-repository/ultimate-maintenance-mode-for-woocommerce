== Description ==
The best plugin to apply maintenance mode to your WooCommerce shops with many features to apply the maintenance mode without the need to shut down the website or hide the products.

= Features =

* Enable - Disable the Maintenance mode easily through WooCommerce Settings Page.
* Use a specific Date to automatically disable the maintenance mode.
* Disable the maintenance mode for some user roles.
* Apply The maintenance mode on some parts of the shop products selected by categories / tags / pages or specific producst by choice.
* An option to redirect some products to other links.
* Display a notice on all WooCommerce pages to let customers know the shop is in maintenance mode.

== Installation ==

1. Upload the plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. The settings page is located at WooCommerce/Settings/Maintenance Mode Tab.
